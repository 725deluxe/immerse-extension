# Immerse — イマージョン学習プレイヤー

YouTubeでイマージョン学習をするためのChrome拡張機能です。  
動画ページを開くと右下にパネルが表示され、**ABリピート・速度変更・学習ログ**が使えます。

---

## こんな人に

- 英語の動画をひたすら繰り返し聴いてイマージョン学習している
- 気になった表現やフレーズを何度も聴き込みたい
- 今日どれくらい学習したか記録したい

---

## 機能

| 機能 | 内容 |
|---|---|
| **ABリピート** | A点・B点を指定した区間だけ無限ループ |
| **全体ループ** | AB未設定のときは動画全体をループ |
| **速度変更** | 0.5× / 0.75× / 1× / 1.25× / 1.5× |
| **ループ回数** | 現在の動画のループ回数を表示 |
| **今日の視聴時間** | その日の合計学習時間を記録 |
| **総ループ数** | 累計ループ数を記録 |

---

## インストール方法

ChromeのWeb Storeには未公開のため、以下の手順で手動インストールします。  
難しい操作はありません。3分でできます。

**1. ファイルをダウンロード**

このページ右上の緑のボタン「**Code**」→「**Download ZIP**」をクリック

**2. ZIPを解凍**

ダウンロードした `immerse-extension-main.zip` を解凍します。

**3. Chromeに読み込む**

1. Chromeのアドレスバーに `chrome://extensions` と入力してEnter
2. 右上の「**デベロッパーモード**」をオンにする
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. 解凍してできたフォルダ（`immerse-extension-main` の中の `immerse-extension`）を選択

**4. 完了**

YouTubeの動画ページを開くと、右下にパネルが表示されます。

## 使い方

<img src="https://via.placeholder.com/600x200?text=パネルのスクリーンショットをここに貼る" alt="パネルのスクリーンショット">

**ABリピートの手順：**

1. 動画を再生し、繰り返したい区間の**始まりで「A点」**をクリック
2. 終わりで**「B点」**をクリック
3. 自動的にA→Bをループし続けます
4. 「クリア」で解除（動画全体のループに戻ります）

**速度変更：**

パネル上部のボタンで切り替えます。YouTubeの標準機能と連動します。

## 動作環境

- Google Chrome（最新版推奨）
- YouTube（youtube.com）のみ対応

---

## 注意事項

- このツールはYouTubeの公式ツールではありません
- 学習データ（視聴時間・ループ数）はお使いのブラウザにのみ保存され、外部には送信されません
- YouTubeの仕様変更により動作しなくなる場合があります
